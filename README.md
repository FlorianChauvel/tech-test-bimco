
## Présentation

Ce projet est un test technique pour l'entreprise Bim&Co(https://www.bimandco.com). Il a été créé avec [Create React App](https://github.com/facebook/create-react-app).

Pour le lancer, il faut tout d'abord installer [NodeJS](https://nodejs.org/en/)
Puis utiliser les commandes suivantes: 
`npm install`
`npm start`

Il possède aussi quelques tests unitaires, pour les executer il faut executer la commande suivante: 
`npm test`

## Technologies utilisées
- React (sans Redux, assez peu de state à gérer)
- [Emotion](https://emotion.sh) (librairie JSS)
- [Enzyme](https://airbnb.io/enzyme/) (facilite les tests unitaires sur des composants React)
